# LinkedIn Post Generator

A simple web-based tool to generate engaging LinkedIn post series from stock analysis or other data.

## Features
- **Clean UI**: A modern interface with a sidebar for post templates and a main editor area.
- **Dynamic Templates**: Pre-configured templates for different types of posts (e.g., "The Hook", "The Deep Dive").
- **Real-time Preview**: See how your post looks as you type.
- **AI-Ready**: Designed to be used in conjunction with AI tools for content generation.

## How to Use
1. **Select a Template**: Choose a post type from the sidebar.
2. **Fill in Details**: Enter the required information (e.g., stock name, key metrics).
3. **Generate**: Click "Generate Post" to create the draft.
4. **Copy**: Use the "Copy to Clipboard" button to grab the text.

## Pro Tip
Take a screenshot of the filled-out form and feed it to an AI model (like ChatGPT or Claude) to generate even more detailed variations or follow-up posts!
